# Changelog

## [1.0.0] - 2025-11-05

### Added
- Initial release with Google Calendar API v3 integration
- Event CRUD operations (create, read, update, delete)
- Recurring event support with RRULE syntax
- Availability checking and FreeBusy queries
- Multiple calendar management
- Attendee and invitation management
- Reminders and notifications
- Google Meet integration
- Event colors and visibility settings
- Import/export functionality (ICS format)
- Time zone support
- OAuth2 authentication flow
